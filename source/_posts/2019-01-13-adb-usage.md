### 下载 ADB 工具

[Download the ADB ZIP file for macOS](https://dl.google.com/android/repository/platform-tools-latest-darwin.zip)

### HTC ONE 开启开发者模式

设置 -> 关于 -> 软件信息 -> 软件版本（点击7次）


### 1 安装 apk 

```shell
adb install apk完整路径）
```

<!-- more -->


### 2 卸载 apk

```shell
adb shell rm data/app/apk-name.apk
```
