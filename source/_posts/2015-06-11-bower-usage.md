#### 1. 安装 `nodejs`
    
```shell
$ brew install nodejs
```

NPM是node程序包管理器。它是捆绑在nodejs的安装程序上的，所以一旦你已经安装了node，NPM也就安装好了。


#### 2. 安装 `bower`
 
```shell
$ npm install -g bower
```


#### 3. 使用 `bower` 安装其它库

```shell
$ bower install jquery
```
